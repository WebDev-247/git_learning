# Calculator (Web)

A simple, modern, and accessible calculator built with vanilla HTML, CSS, and JavaScript. No frameworks required.

## Features

- Basic operations: addition, subtraction, multiplication, division
- Keyboard support (0-9, ., + - * /, Enter/=, Backspace, Escape)
- Safe evaluation without `eval`
- Responsive, clean UI

## Getting Started

Open `index.html` directly in your browser, or run a local static server:

```bash
cd /workspace/calculator
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

## Files

- `index.html`: App markup
- `styles.css`: App styles
- `app.js`: Calculator logic and keyboard handling
